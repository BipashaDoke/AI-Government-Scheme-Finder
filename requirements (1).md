# Project Requirements Document

## Project Title
AI Government Scheme Finder

## Objective
To help users identify government schemes they are eligible for using AI-based analysis.

## Functional Requirements
- User authentication
- User profile data collection
- AI-based eligibility analysis
- Display eligible government schemes
- Scheme details and official links

## Non-Functional Requirements
- Simple and user-friendly UI
- Fast response time
- Secure handling of user data

## Tech Stack
- Frontend: React
- Backend: Python (Flask)
- Database: MongoDB
- AI Integration: Kiro AI / Gemini API

## Tools & Platform
- Kiro IDE
- VS Code
- GitHub
- Web Browser
